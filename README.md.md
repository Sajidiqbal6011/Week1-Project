Project Title: Cloud Basics Dataset

Use Case
This dataset is built for a Cloud Basics Explainer AI Assistant that helps fresh computer science students easily understand cloud computing terminology.

Target Users
1st and 2nd semester CS students who are completely new to cloud platforms and need logical answers to clear their foundational concepts.

Project Files
README.md: This documentation file.
cleaned_examples.jsonl: The main file containing all human-verified question and answer pairs.
train.jsonl: 80% split of the cleaned data used for training.
test.jsonl: 20% split of the cleaned data used for testing.
review_notes.md: Notes tracking data verification steps and quality checks.
